## デザイン方針

### 参考サイト
1. **https://kyafuk.github.io/khirashima/** - シンプル・テキスト中心・NEWS表示
2. **https://shobuzako-kensuke.github.io/** - 単一カラム・日英併記
3. **https://tomyoshida.github.io/** - 落ち着いた配色・学術リンク
4. **https://naomichiyutani.github.io/HP/** - 洗練された配色・固定ヘッダー・セクションボーダー

### 採用するデザイン要素
- **レイアウト**: シングルカラム、中央揃え（max-width: 960px）
- **配色**:
  - 背景: #f7f8fb（薄青灰色）
  - テキスト: #202533（濃紺）
  - アクセント: #28527a（深青）
  - サブテキスト: #5a6072（グレー）
- **フォント**:
  - 見出し: "Merriweather", "Yu Mincho", serif
  - 本文: "Source Sans 3", "Hiragino Sans", sans-serif
- **特徴**:
  - 固定ヘッダーナビゲーション
  - セクション左に3pxボーダー
  - 角丸カード（8px）
  - 控えめなシャドウ
  - レスポンシブ対応（720px以下でハンバーガーメニュー）

## GitHub Pages 設定手順
1. https://github.com/Tenri1198 にリポジトリ作成（tenri1198.github.io）
2. index.html, style.css, 画像ファイルをプッシュ
3. Settings → Pages → Source: main branch
4. https://tenri1198.github.io/ で公開
